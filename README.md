## Hi, I'm Megan Schott 

I build serverless apps and AI / MCP tools on AWS — plus home-automation tinkering and the occasional iOS app.

## Projects

* **[meal-planner-mcp](https://github.com/illinigirl/meal-planner-mcp)** — self-contained MCP server that plans a week of meals from a local recipe library: optimizes ingredient overlap, reuses leftovers, and avoids recent repeats, then emits a shopping list + Markdown plan. Clone-and-run.
* **[MagicMonitor](https://github.com/illinigirl/MagicMonitor)** — live Walt Disney World ride-status dashboard + Pushover alerts (Lambda · DynamoDB · EventBridge · Amplify), with an MCP-exposed agentic trip planner.
  * **Live:** https://magicmonitor.megillini.dev
* **[decades-trivia](https://github.com/illinigirl/decades-trivia)** — adaptive decade-based trivia on AWS (Bedrock · Lambda · DynamoDB). Questions are generated by Claude, fact-checked by two-model consensus, with Wikipedia RAG behind the review-facts feature.
* **[python-turtle-workshop](https://github.com/illinigirl/python-turtle-workshop)** — zero-install, browser-based workshop that teaches kids real Python with a turtle and an optional AI helper. Runs entirely in the browser via Pyodide.
* **[FrameShuffle](https://github.com/illinigirl/FrameShuffle)** — rotate your own art & photos on a Samsung Frame TV from a local web UI: shuffle, folder / color filtering, and battle-tested 2024-model quirk handling. No cloud, no account.
* **[TripSplit](https://github.com/illinigirl/TripSplit)** — native iOS app (SwiftUI) for splitting shared trip expenses: flexible per-item splitting and a graph-based algorithm that settles the group in the fewest possible payments.
